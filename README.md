# Abyss Survivor

## v6.0 이벤트 런 / 변칙 규칙

- 이벤트 런 선택을 추가했습니다. 로비에서 일반, 피의 밤, 보석 폭풍, 약자의 계약, 침묵의 전장, 심연 폭주 중 하나를 선택할 수 있습니다.
- 이벤트는 기존 클래스, 무기, 구역, 도전, 심연 단계와 함께 적용되는 변칙 규칙입니다.
- 이벤트별로 적 체력/피해/속도, 스폰 속도, 적 수 상한, 보석 보상, 처치 회복, 유물 선택/효과, 최종 보상 배율이 달라집니다.
- 저장 데이터에 `selectedEventId`가 추가되며, 기존 저장 데이터는 `normal` 이벤트로 자동 보정됩니다.
- 결과 화면에는 선택한 이벤트와 최종 보상 배율이 표시됩니다.
- `test-runner.html`에 v6 이벤트 데이터, 저장 fallback, modifier 안정성, 보상/전투 반영, 이벤트 선택 UI 검수를 추가했습니다.

## v5.1 밸런스/룰/임무 개선

- 번개 사슬 계열의 첫 대상 사거리, 연쇄 사거리, 기본 피해를 크게 낮추고 화면 밖 적은 대상에서 제외합니다.
- 클리어 조건은 3분 생존으로 고정했습니다. 보스 처치는 런을 끝내지 않고 심연 폭주 웨이브를 시작합니다.
- 보스 처치 후 남은 시간 동안 스폰 간격, fast/tank 비중, 정예 확률, 적 수 상한이 강화됩니다.
- 보스 접촉 피해는 tick 방식으로 적용되며, 회전 칼날/피의 낫 같은 근접 무기는 보스 접촉 피해를 일부 줄입니다.
- 기존 1회성 임무는 매 런 3개가 생성되는 런 임무로 바뀌었고 결과 화면에서 완료 여부와 보상을 표시합니다.
- `test-runner.html`에 v5.1 번개 너프, 3분 클리어, 보스 처치 후 런 지속, 보스 접촉 tick, 런 임무 보상 중복 방어 검수를 추가했습니다.

`Abyss Survivor`는 모바일 세로 화면 기준의 싱글 플레이 생존 로그라이트 웹게임입니다. 순수 HTML, CSS, JavaScript만 사용하며 외부 라이브러리, ES Module, npm, 번들러 없이 GitHub Pages 루트에서 바로 실행됩니다.

## 핵심 흐름

- 클래스, 구역, 도전 모드, 영구 성장을 선택합니다.
- Canvas를 드래그하거나 WASD/방향키로 이동합니다.
- 플레이어는 가장 가까운 적을 자동 공격합니다.
- 경험치 보석을 획득해 레벨업 능력, 진화, 유물을 선택합니다.
- 3분 동안 생존하면 런을 클리어하며, 보스 처치는 심연 폭주 웨이브를 여는 중간 목표입니다.
- 결과에 따라 심연 조각을 얻고 다음 런의 영구 성장에 사용합니다.

## 주요 시스템

- 클래스 3종: 방랑자, 수호자, 추격자
- 구역 3종: 균열 입구, 침묵 늪지, 심연 핵
- 도전 모드 4종: 일반, 굶주린 심연, 유리 생존자, 빠른 침식
- 레벨업 능력과 진화 스킬
- 유물 8종과 빌드 조합 보너스 3종
- 심연 조각 보상과 영구 성장 3종
- 구역별 보스와 시간대별 웨이브
- localStorage 기반 저장 데이터 호환

## 조작 방법

- 모바일: Canvas를 누르고 드래그해 이동
- PC 마우스: Canvas 드래그 이동
- PC 키보드: WASD 또는 방향키 이동
- 하단 `Ⅱ` 버튼: 일시정지
- 하단 `↻` 버튼: 재시작

## 실행 방법

저장소 루트의 `index.html`을 브라우저에서 열면 실행됩니다. GitHub Pages에서는 저장소 루트를 그대로 배포 기준으로 사용합니다.

## 테스트 방법

저장소 루트의 `test-runner.html`을 브라우저에서 열면 주요 모듈, 저장 데이터 fallback, 런 생성, 반복 업데이트, 렌더 호출, 로비 모달, 사거리 제한 값을 자동 점검합니다. 결과는 화면에 `통과` 또는 `실패`로 표시됩니다.

## 파일 구조

- `index.html`: 화면 구조와 일반 `<script src="">` 로딩
- `style.css`: 모바일 세로형 UI 스타일
- `data.js`: 고정 데이터, 클래스, 구역, 능력, 유물, 도전 모드
- `state.js`: 런 상태와 저장 데이터 관리
- `game.js`: 게임 업데이트와 전투 규칙
- `render.js`: Canvas 도형 렌더링
- `ui.js`: DOM, HUD, 오버레이, 입력 처리
- `main.js`: 초기화와 게임 루프
- `test-runner.html`: 브라우저 기반 기본 검수 페이지

## v3.1 무기 성장 체감 개선

- 픽토그램 박스와 글자 크기를 키우고 카드/버튼 중앙 정렬을 보정했습니다.
- 심연 탄환 외 시작 무기도 공격력, 공격 속도, 투사체 속도, 투사체 크기, 수량 계열 강화의 영향을 받습니다.
- 회전 칼날은 성장에 따라 칼날 수, 크기, 회전 속도, 피해가 증가합니다.

- 공허 지뢰는 폭발 범위, 설치 수, 피해가 성장하며 폭발/심연 계열 빌드 보너스를 받습니다.
- 영혼 파동은 범위, 추가 파동 수, 피해가 성장하며 일반 강화와 태그 보너스를 받습니다.
- localStorage 저장 구조는 변경하지 않았고, 무기별 성장값은 현재 런 안에서만 관리합니다.
