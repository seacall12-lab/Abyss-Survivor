Abyss Survivor v0.1

v0.1 목표
- 모바일 세로형 화면에서 실제로 플레이 가능한 최소 생존 게임 완성
- 순수 HTML, CSS, JavaScript만 사용
- GitHub Pages 루트 실행 구조 유지

구현된 기능
- 플레이어 이동
- 모바일 터치, PC 마우스 드래그, WASD/방향키 입력
- 자동 공격
- 적 생성과 플레이어 추적
- 투사체 충돌과 적 처치
- 경험치 보석 생성과 획득
- 레벨업 능력 3개 선택
- 일시정지와 재시작
- 게임 오버
- 보스 등장과 클리어
- 생존 기록 localStorage 저장

아직 제외된 기능
- 장비
- 뽑기
- 무기 진화
- 도감
- 업적
- PWA
- 사운드
- 복잡한 보스 패턴
- 온라인 기능

다음 v0.2 예정 작업
- 무기 3종 추가
- 적 3종 추가
- 기본 밸런스 조정

v0.1 안정화
- 모바일 터치 입력이 끊기거나 남지 않도록 포인터 해제 처리를 보강
- WASD 입력이 브라우저 포커스 이탈 후 남지 않도록 초기화 처리
- 레벨업, 게임 오버, 클리어 상태 전환 직후 전투 로직이 계속 진행되지 않도록 보정
- 보스 등장 시 적 수 제한에 막혀 보스가 생성되지 않는 문제 방지
- 보석 획득 반경 주변에서 보석이 조금 더 자연스럽게 모이도록 조정
- Canvas 모바일 스크롤 방지와 화면 폭 제한 보강

v0.3 통합 업데이트
- 진화 능력 4종 추가: 관통 파편, 분열 파편, 회전 보호막, 심연 폭발
- 레벨 4 이상부터 진화 능력이 선택지에 등장하도록 개선
- 이미 획득한 진화 능력과 최대 레벨에 도달한 일반 능력은 선택지에서 제외
- 능력 카드에 강화/진화 구분 표시 추가
- 보스 돌진, 소환, 오라 패턴 추가
- 진화 능력과 보스 패턴의 Canvas 도형 효과 추가
- v0.3 기준 투사체 수와 전투 밸런스 2차 조정

v0.4~v0.5 통합 업데이트
- 클래스 3종 추가: 방랑자, 수호자, 추격자
- 구역 3종 추가: 균열 관문, 늪지 가장자리, 심연 핵
- 구역별 보스 3종 연결: 균열 감시자, 늪지 수호자, 핵 포식자
- 처치 수, 생존 시간, 클리어 여부에 따라 심연 조각 보상 지급
- 심연 조각으로 구매하는 영구 성장 3종 추가: 생명력, 공격력, 성장력
- 시작 화면에서 클래스, 구역, 영구 성장을 선택할 수 있도록 UI 추가
- 시간대별 웨이브 설정으로 적 생성 간격과 타입 비중 조정
- 저장 데이터 version 2로 확장하고 기존 저장 데이터와 호환되도록 보정

v0.7~v0.9 통합 업데이트
- 유물 시스템 추가
- 유물 8종 추가: 피의 핵, 날카로운 심장, 사냥꾼의 눈, 탐식의 파편, 검은 날개, 부서진 방패, 공허의 손, 심연의 계약
- 빌드 조합 보너스 추가: 탄환 빌드, 근접 생존 빌드, 폭발 빌드
- 도전 모드 4종 추가: 일반, 굶주린 심연, 유리 생존자, 빠른 침식
- 도전 모드별 보상 배율과 런 시작 modifier 적용
- 런 결과 요약 개선
- 저장 데이터 version 3으로 확장하고 기존 version 2 저장 데이터와 호환되도록 보정
- 모바일 시작 화면 UI 밀도 정리
- v0.9 기준 보상, 유물, 도전 모드 밸런스 정리
