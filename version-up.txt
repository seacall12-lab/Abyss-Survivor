Abyss Survivor v10.0

v10.0 최종 안정화 / 가벼운 PWA / 아이콘 마감
- 새 대형 게임 기능 추가 없이 현재 구현된 v8 지도와 v9 엔드게임 흐름을 기준으로 안정화
- `index.html`의 기본 표시 문구와 PWA 메타/manifest/apple touch icon 연결 정리
- `manifest.json`, `service-worker.js`, `icons/icon-192.png`, `icons/icon-512.png` 추가
- service worker 캐시명을 `abyss-survivor-v10.0.0`으로 명확히 지정
- service worker 등록 실패가 게임 실행 실패로 이어지지 않도록 `main.js`에서 경고만 출력
- 저장 데이터에 `settings` fallback 추가, 기존 저장 데이터는 삭제하지 않고 보정
- 런타임 player/enemy/projectile/gem/damageText 주요 숫자에 NaN/Infinity 방어 추가
- 생존 모드와 보스 러시 지도 선택 흐름 분리 검수
- v8 mapChoice 상태 전환, 선택 중 전투 업데이트 정지, 결과 경로 표시 검수
- v9 최종 보스/변종 보스/첫 클리어 보상 중복 지급 방어 검수
- 클래스, 무기, 보조무기, 구역, 이벤트, 유물, 보스, 결과 화면 아이콘 fallback 검수
- `test-runner.html`을 v10 최종 검수 항목 중심으로 재정리
- `README.md`, `PROJECT_SPEC.md`, `development-history.txt`를 실제 구현 기준으로 갱신
